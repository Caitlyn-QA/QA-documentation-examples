# Pagination Test Cases

## TC-007 Pagination Controls Display
### Precondition
User is on the Product Overview page.

### Steps
1. Open the Product Overview page.
2. Observe the pagination controls.

### Expected Result
Pagination controls are displayed when multiple pages of products are available.

## TC-008 Navigate to Another Page
### Precondition
User is on the Product Overview page.
Pagination controls are available.

### Steps
1. Click a page number in the pagination controls.
2. Observe the displayed products.
   
### Expected Result
The product list is updated to display products from the selected page.
The selected page is indicated as active.
