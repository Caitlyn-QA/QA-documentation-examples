# QA Documentation Examples

This repository contains examples of common QA documentation created against the Practice Software Testing application.

The goal of this project is to demonstrate how requirements can be translated into test plans, test cases, bug reports, and traceability documentation.

## Included Documentation
- Test Plan
- Manual Test Cases
- Bug Reports
- Requirements Traceability Matrix

## Application Under Test
Practice Software Testing

## Focus Areas:
- Product Search
- Category Filtering
- Pagination
- Product Navigation
